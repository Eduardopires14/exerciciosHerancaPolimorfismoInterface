# Exercícios de Herança, Polimorfismo e Interface

Este repositório contém exercícios práticos em **Java** para treinar conceitos fundamentais da Programação Orientada a Objetos (POO):

- **Herança**: reutilização de código e especialização de classes.
- **Polimorfismo**: capacidade de objetos assumirem diferentes formas, permitindo flexibilidade no uso de métodos.
- **Interfaces**: definição de contratos que classes podem implementar, garantindo consistência e abstração.
